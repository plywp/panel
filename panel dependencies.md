# Panel System Dependencies

- Node.js (runtime for `vite`/SvelteKit and `npm run ...` scripts)
- A JavaScript package manager: `npm`, `pnpm`, or `yarn` (used by the install and dev scripts)
- `bun` (used by the `auth:generate` script)
- MySQL or MariaDB server (used by `mysql2` and `drizzle` tooling)
